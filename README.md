
# alanator

бесплатный архиватор написанный на языке Python с поддержкой zip, 7z, tar и собственного формата .aln

## возможности
- архивация zip, 7z, tar
- собственный формат .aln
- удобный интерфейс как у peazip
- настройки темы и языка
 
## установка
1. скачай zip-архив из раздела releases
2. расспакуй где угодно.
3. запусти alanator.exe и пользуйся

## для разработчиков
python 3.12+
customtkinter

## навигация
чтобы создать архив
1. зайти в alanator
2. перейти в любую папку с файлами, не с папками, а именно с файлами
3. нажать кнопку "Новый архив"
4. по желанию поставить пароль
5. нажать OK
## известные баги 
проект находится в разработке возможны ошибки
1. после закрытия окна просмотра архива может появиться окно аланатора. просто закройте его.
2.  при открытии .aln с паролем может не запросить пароль. работаем над исправлением.
3. zip без пароля может открываться с небольшой задержкой.
