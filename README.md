# alanator
 
бесплатный архиватор с поддержкой zip, 7z, tar и собственного формата .aln

## возможности
- архивация zip, 7z, tar
- собственный формат .aln
- удобный интерфейс как у peazip
- настройки темы и языка

## установка
1. скачай alanator.exe из раздела releases
2. запусти и пользуйся

## для разработчиков
python 3.12+
pip install customtkinter

- **Бесплатно** — никаких "купите лицензию"

- **Просто** — интерфейс как у PeaZip, понятно с первого раза
