# 📦 alanator

![Python](https://img.shields.io/badge/Python-3.12+-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-success)
![License](https://img.shields.io/badge/License-MIT-green)
![Version](https://img.shields.io/badge/Version-v0.9.0-orange)

**alanator** — бесплатный архиватор с современным графическим интерфейсом, написанный на **Python**.

Программа поддерживает популярные форматы архивов **ZIP**, **7Z**, **TAR**, а также собственный формат **`.aln`**.

---

# ✨ Возможности

* 📦 Создание архивов ZIP
* 📦 Создание архивов 7Z
* 📦 Создание архивов TAR
* 📦 Создание архивов ALN
* 🔐 Защита архивов паролем
* 👀 Просмотр содержимого архивов
* 🎨 Светлая и тёмная темы
* 🌍 Поддержка нескольких языков
* 🖥️ Современный интерфейс на CustomTkinter
* 📂 Ассоциация файлов `.zip`, `.7z`, `.tar`, `.aln`

---

# 📥 Установка

1. Откройте раздел **Releases**.
2. Скачайте последнюю версию программы.
3. Распакуйте архив.
4. Запустите **alanator.exe**.

---

# ⚠️ SmartScreen Windows

При первом запуске Windows может показать предупреждение **Microsoft Defender SmartScreen**.

Это происходит потому, что приложение пока не имеет цифровой подписи.

Чтобы продолжить:

**Подробнее → Выполнить в любом случае**

# 🚀 Быстрый старт

## Создание архива

1. Запустите alanator.
2. Перейдите в папку с файлами.
3. Нажмите **«Новый архив»**.
4. Выберите формат архива.
5. При необходимости установите пароль.
6. Нажмите **OK**.
